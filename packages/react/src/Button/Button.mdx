Buttons allow users to take actions and make choices with a single tap.

<Canvas of={ButtonStories.ButtonStory} />

## Props

### Handling Clicks

Buttons accept an `onClick` handler that is applied to the button element.

<Source
  dark
  code={`<Button
  onClick={() => {
    alert('clicked');
  }}
>
  Click me
</Button>`}
/>

### Variants

The Button comes with five variants: `plain`, `outline`, `soft`, `solid` (default), and `achromatic`.

<Canvas
  of={ButtonStories.Variants}
  source={{
    language: "tsx",
    code: `
<Button variant="plain">plain</Button>
<Button variant="outline">outline</Button>
<Button variant="soft">soft</Button>
<Button variant="solid">solid</Button>
<Button variant="achromatic">achromatic</Button>
`,
  }}
/>

### Sizes

Use the `size` prop to change the size of the button.

<Canvas
  of={ButtonStories.Sizes}
  source={{

<Button size="md">Button</Button>
<Button size="sm">Button</Button>
<Button size="xxs">Button</Button>`,
  }}
/>

### Colors

Button supports the palette included in Ripley. Use the color prop to change the color.

<Canvas
  of={ButtonStories.Colors}

### Links and Polymorphism

Use the `as` prop to create a hyperlink. Pass an `a` tag to the prop to change the root element.

<Canvas of={ButtonStories.Link} source={{ code: `<Button as="a">I'm an anchor element</Button>` }} />

### Button with Icons and Labels

If you want to enhance the UX of your button with an icon, compose them within the `children` of the button:

<Canvas
  of={ButtonStories.ButtonWithIconAndLabel}
  source={{
    code: `<Button variant="outlined">
  <>
    <DeleteIcon />
    Delete
  </>
</Button>
<Button variant="solid">
  <>
    Send 
    <SendIcon />
  </>
</Button>`,
  }}
/>

## Anatomy

The Button is composed of a single root `<button>` element that wraps its contents.

<Source
  dark
  code={`<button>
  {children}
</button>`}
/>
